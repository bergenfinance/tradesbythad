Data Management

Export

Download JSON files to update your site's data. Place them in the data/ folder.

Import

Import JSON files to replace current data. This will overwrite existing data in the admin (not on disk until exported).

Direct File Access

In Chrome/Edge, you can connect to your project folder for direct file writes. Click "Connect Folder" in the header and select your project root. Saves will then write directly to data/ without downloading.
